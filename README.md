# DevJobs — Bootcamp Reinforcement Project

DevJobs is a small practice project built during a reinforcement bootcamp. It is a static front-end prototype that demonstrates layout, responsive grid, and basic UI patterns for a developer jobs site.

## Project status

- Learning / prototype
- Not production-ready
- Assets: HTML + CSS only

## How to view

Open the main demo in your browser:

- [00-html-css/index.html](00-html-css/index.html)

## Project structure

- README.md — this file
- 00-html-css/
  - [index.html](00-html-css/index.html) — main demo page
  - [styles.css](00-html-css/styles.css) — styles for the demo
- 01-javascript/ — exercises and JS tasks
- 02-react/ — React exercises
- 04-node/ — Node.js exercises
- 05-typescript/ — TypeScript exercises
- 06-sql/ — SQL exercises
- 07-ci-cd/ — CI/CD examples
- 08-docker/ — Docker examples

## Goals

- Reinforce HTML & CSS fundamentals (layout, variables, icons, accessibility).
- Practice progressive enhancement and semantic markup.
- Prepare small components to be ported to JS/React later.

## Technologies

- HTML5
- Modern CSS (CSS variables, grid, flexbox)
